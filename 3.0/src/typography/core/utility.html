常用工具样式

浮动

  • x-clear: 清除浮动
  • x-left: 左浮动
  • x-right: 右浮动

居中

  • x-middle: 垂直居中
  • x-center: 水平居中

隐藏

  • x-hide: 隐藏元素（display:none)
  • x-hide-small: 只在小屏幕（即手机）隐藏
  • x-hide-medium: 只在非小屏幕（即平板、电脑）隐藏
  • x-hide-large: 只在大屏幕（即电脑）隐藏
  • x-show-small: 只在小屏幕（即手机）显示
  • x-show-medium: 只在非小屏幕（即平板、电脑）显示
  • x-show-large: 只在大屏幕（即电脑）显示

文本

  • x-ellipsis: 文本超出区域后显示 ...，而不是自动换行或隐藏
  • x-break-word: 强制文本换行，避免长单词不换行。适合中文排版使用

其它

  • x-blank: 提供 1em(14px) 高的空白行
  • x-scrollable-horizontal: 超出内容后显示滚动条
