界面开发设计模式

界面开发一直需要大量的工作，主要有以下原因：用户操作的不确定性、代码执行环境（如浏览器、操作系统、硬件）的不确定性、越来越高的用户体验要求、界面和数据、逻辑交互能力弱等。因此诞生了 AngularJs、ReactJs 等用于开发界面软件的框架。

TealUI 有意创造自主的界面开发模式，这个模式的特点有：

  • 尽量保持语言原始特性。
  • 坚持面向对象和面向组件。
  • 坚持用最少的代码做常见的需求。
  • 不实现不常用功能，以精简代码量。
