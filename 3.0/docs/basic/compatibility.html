浏览器兼容性

浏览器兼容性问题是前端开发所面对的最头疼问题。TealUI 作为国产前端框架，以国内浏览器市场占有率为基础，定制了以下了浏览器分类。

对于其它国产浏览器（如 UC、360浏览器、QQ浏览器、百度浏览器等），级别均根据其内核模式定义。

兼容性原则
  1. A 类浏览器：完全支持。
  2. B 类浏览器：提供可选补丁支持，但不能确保和一级浏览器有完全相同的用户体验。
  3. C 类浏览器：不测试其兼容性，但会提供小部分补丁。
  4. X 类浏览器：不测试其兼容性，理论上也不会提供补丁。

分辨率兼容性

所有组件优先考虑移动平台，同时为 PC 平台添加自适应补丁。如果组件只用于移动平台，可以保证最少的代码量。

前缀 分辨率宽度 描述
-smaller 0-640px 手机
-small 641px-1024px 平板
-large 1025px-1280px 电脑
-larger 1281px-1440px 电视
-largest 1441px-1920px 高清电视、投影仪

TealUI 考虑前端应用场景，只提供区分手机和非手机平台的样式。
