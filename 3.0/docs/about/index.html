TealUI 目标

TealUI 的目标是打造一个完整的开源前端代码库。 TealUI 提供的组件功能齐全、风格统一、性能优秀、代码轻量。 用户无需再去网上搜索各类开源项目，也无需面对风格不一的各种文档。 一般地，只要学会一个组件的用法，就能马上掌握所有同类组件的用法。

TealUI 历史

  • 2010: 由于业务需要，JPlusUI 诞生。最初目的是为项目提供实用的 UI 组件库。所有组件基于 jQuery 开发。
  • 2011: 组件库扩展到了 200 多个组件，并不再依赖 jQuery，所有组件兼容 IE6。
  • 2012: 参考 BootStrap 的界面进行了全部重新设计，并发布了第一个完整版。
  • 2013: 根据用户的反馈修复了多个 BUG 并发布了第二个稳定版。此后 JPlusUI 停止更新。
  • 2015: JPlusUI 更名 TealUI，对所有组件进行重写，抛弃了 IE6-7 兼容，改为移动优先策略。
