    ☍基本用法

    __root__,

    ☍API

    函数 描述
    ⅏ ⮞ addToHomeScreen(options)(options:any):void

    对于 IOS/Android 用户，弹出添加到主屏幕的浮层。

    参数 类型 描述 默认值
    options* any

    可选的配置信息。

    		 —

    返回值

    类型：void

    说明

    请确保必须在 DomReady 中调用本函数。
