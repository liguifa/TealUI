    ☍API

    函数 描述
    ⅏ ⮞ bresenham(x0, y0, x1, y1, ...)(x0:number, y0:number, x1:number, y1:number, callback?:function):object[]

    创建一个贝塞尔曲线。

    参数 类型 描述 默认值
    x0* number

    起始点的水平坐标。

    		 —
    y0* number

    起始点的垂直坐标。

    		 —
    x1* number

    结束点的水平坐标。

    		 —
    y1* number

    结束点的垂直坐标。

    		 —
    callback ((x: number, y: number) => void) | undefined

    所有生成的点的回调函数。

    		 —

    返回值

    类型：{ x: number; y: number; }[]

    如果未提供回调函数则返回生成的点坐标。
