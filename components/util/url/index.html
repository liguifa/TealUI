    ☍URL 处理
    util/url
  • 复制 import "util/url"
  • 复制 require("util/url")
  • 复制 "util/url"
    •

    提供解析和格式化 URL 的工具函数。

    本组件利用原生 JavaScript 解析地址。利用浏览器自带的 <a> 标签实现解析见URL 处理。

    ☍API
