    提供 JavaScript 内置对象 Number 的扩展接口。

    ☍API

    函数 描述
    ⅏ ⮞ isNumber(obj)(obj:any):boolean

    判断一个对象是否是数字。

    参数 类型 描述 默认值
    obj* any

    要判断的对象。

    		 —

    返回值

    类型：boolean

    如果是数字则返回 true，否则返回 false。

    示例

    ❐isNumber(7) // true

    判断一个对象是否是数字。

    判断一个数字是否是整数。

    参数 类型 描述 默认值
    num* number

    要判断的数字。

    		 —

    返回值

    类型：boolean

    如果是整数则返回 true，否则返回 false。

    示例

    ❐isInteger(7) // true

    判断一个数字是否是整数。
    ⅏ ⮞ limit(num, min, max)(num:number, min:number, max:number):number

    确保指定数值在指定区间内。

    参数 类型 描述 默认值
    num* number

    要判断的值。

    		 —
    min* number

    允许的最小值。

    		 —
    max* number

    允许的最大值。

    		 —

    返回值

    类型：number

    如果 num 超过 min 和 max，则返回临界值，否则返回 num。

    示例

    ❐limit(1, 2, 6) // 2

    确保指定数值在指定区间内。
    ⅏ ⮞ random(min, max)(min:number, max:number):number

    返回指定范围的随机整数值。

    参数 类型 描述 默认值

    最小的整数值。

    		 —
    max* number

    最大的整数值。

    		 —

    返回值

    类型：number

    返回结果值。其值满足 min <= 返回值 < max。

    示例

    ❐random(2, 6)

    返回指定范围的随机整数值。

    ⅏ ⮞ format(num, ...)(num:number, format?:string):string

    格式化数字为字符串。

    参数 类型 描述 默认值
    num* number —
    format string | undefined

    格式字符串。 格式字符串中的元字符会被替换，支持的元字符有：

    元字符 意义 示例
    0 补0 000:012, 0.00: 2.00
    # 补空 ###:12, 0.##: 2
    + 补正负号 +0: +0
    . 小数点 0.00: 0.00
    , 追加逗号 0.00: 0.00
    		—

    返回值

    类型：string

    返回格式化后的字符串。

    示例

    ❐format(1, "000.00") // 001.00
    ❐format(1.2, "0.00") // 1.20
    ❐format(1.2, "#.##") // 1.2
    ❐format(1.235, "#.##") // 1.24
    ❐format(1235, "###,###") // 1,235

    格式化数字为字符串。
