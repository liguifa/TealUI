    ☍API

    函数 描述
    ⅏ ⮞ formatTimeToChinese(date, ...)(date:Date, now?:Date):string

    格式化时间为 刚刚/N 秒(分钟/小时/天)前 格式。

    参数 类型 描述 默认值
    date* Date

    要格式化的日期。

    		 —
    now Date

    指定当前时间。

    		 new Date()

    返回值

    类型：string

    返回时间字符串。如果时间在 5 秒内，返回刚刚，否则返回 N 秒(分钟/小时/天)前 格式。

    示例

    ❐formatTimeToChinese(new Date())

    格式化时间为 刚刚/N 秒(分钟/小时/天)前 格式。
