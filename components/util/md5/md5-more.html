    MD5 加密算法扩展。

    ❐<input type="text" id="input" style="width: 20em;" />
<button onclick="input.value = hmacMd5(input.value, 'key')">HMAC MD5 加密</button>
<button onclick="input.value = base64Md5(input.value)">base64 Md5 加密</button>
<button onclick="input.value = base64HmacMd5(input.value, 'key')">hmacMd5 &amp; HMAC Md5 加密</button>
    ⏷

    ☍API

    函数 描述
    ⅏ ⮞ base64Md5(value)(value:string):string

    使用 MD5 和 Base64 算法加密指定字符串。

    参数 类型 描述 默认值
    value* string

    要计算的字符串。

    		 —

    返回值

    类型：string

    返回加密后的字符串，其中只包含小写字母。

    示例

    ❐base64Md5("abc") // "kAFQmDzST7DWlj99KOF/cg"

    使用 MD5 和 Base64 算法加密指定字符串。
    ⅏ ⮞ hmacMd5(value, key)(value:string, key:string):string

    使用 HMAC-MD5 算法加密指定字符串。

    参数 类型 描述 默认值
    value* string

    要计算的字符串。

    		 —
    key* string

    加密的密钥。

    		 —

    返回值

    类型：string

    返回加密后的字符串，其中只包含小写字母。

    示例

    ❐hmacMd5("abc", "key") // "d2fe98063f876b03193afb49b4979591"

    使用 HMAC-MD5 算法加密指定字符串。
    ⅏ ⮞ base64HmacMd5(value, key)(value:string, key:string):string

    使用 HMAC-MD5 和 Base64 算法加密指定字符串。

    参数 类型 描述 默认值
    value* string

    要计算的字符串。

    		 —
    key* string

    加密的密钥。

    		 —

    返回值

    类型：string

    返回加密后的字符串，其中只包含小写字母。

    示例

    ❐base64HmacMd5("abc", "key") // "0v6YBj+HawMZOvtJtJeVkQ"

    使用 HMAC-MD5 和 Base64 算法加密指定字符串。
