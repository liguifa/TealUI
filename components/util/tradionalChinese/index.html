    ☍简繁体转换
    util/tradionalChinese
  • 复制 import "util/tradionalChinese"
  • 复制 require("util/tradionalChinese")
  • 复制 "util/tradionalChinese"

    简繁体中文转换。

    ❐<input type="text" id="input" placeholder="输入中文" value="简体" />
<button onclick="input.value = toTradionalChinese(input.value)">转繁体</button>
<button onclick="input.value = toSimpleChinese(input.value)">转简体</button>
    ⏷
    实现原理

    在源码中有一个繁简字检索表，函数会检索此表得到对应繁体或简体。

    ☍API

    函数 描述
    ⅏ ⮞ toSimpleChinese(value)(value:string):string

    将繁体中文转为简体中文。

    参数 类型 描述 默认值
    value* string

    要转换的中文。

    		 —

    返回值

    类型：string

    返回转换后的中文。

    示例

    ❐toSimpleChinese("簡体") // "简體"

    将繁体中文转为简体中文。
    ⅏ ⮞ toTradionalChinese(value)(value:string):string

    将简体中文转为繁体中文。

    参数 类型 描述 默认值
    value* string

    要转换的中文。

    		 —

    返回值

    类型：string

    返回转换后的中文。

    示例

    ❐toTradionalChinese("简體") // "簡体"

    将简体中文转为繁体中文。
