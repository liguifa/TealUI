    ☍Base64 编码
    util/base64
  • 复制 import "util/base64"
  • 复制 require("util/base64")
  • 复制 "util/base64"
    •

    提供 Base 64 编码和解码实现。

    ❐<input type="text" id="input" placeholder="输入任意内容" />
<button onclick="input.value = encodeBase64(input.value)">Base64 编码</button>
<button onclick="input.value = decodeBase64(input.value)">Base64 解码</button>
<button onclick="input.value = btoa(input.value)">Base64 编码(仅支持英文)</button>
<button onclick="input.value = atob(input.value)">Base64 解码(仅支持英文)</button>
    ⏷
    提示

    除了 IE6-8 其它浏览器已内置 btoa/atob，但其不支持中文。

    ☍API

    函数 描述
    ⅏ ⮞ encodeBase64(value)(value:string):string

    对指定的字符串进行 Base64 编码(支持中文)。

    参数 类型 描述 默认值
    value* string

    要转换的字符串。

    		 —

    返回值

    类型：string

    返回转换后的字符串。

    示例

    ❐encodeBase64("中文") // "5Lit5paH"

    对指定的字符串进行 Base64 编码(支持中文)。
    ⅏ ⮞ decodeBase64(value)(value:string):string

    对指定的字符串进行 Base64 解码(支持中文)。

    参数 类型 描述 默认值
    value* string

    要转换的字符串。

    		 —

    返回值

    类型：string

    返回转换后的字符串。

    示例

    ❐decodeBase64("5Lit5paH") // "中文"

    对指定的字符串进行 Base64 解码(支持中文)。
