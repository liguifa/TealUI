    提供 JavaScript 内置对象 RegExp 的扩展接口。

    另参考
    • 常用正则表达式库

    ☍API

    函数 描述
    ⅏ ⮞ parse(value, ...)(value:string, flags?:string):RegExp

    解析字符串并创建新的正则表达式。

    参数 类型 描述 默认值
    value* string

    要解析的字符串。字符串的正则元字符将匹配字符本身。

    		 —
    flags string | undefined

    正则表达式标记。可以是 "gmiu" 中任意字符组合。

    		 —

    返回值

    类型：RegExp

    返回一个新正则表达式对象。

    示例

    ❐parse("\\s") // /\s/

    解析字符串并创建新的正则表达式。
    ⅏ ⮞ isRegExp(obj)(obj:any):boolean

    判断一个对象是否是正则。

    参数 类型 描述 默认值
    obj* any

    要判断的对象。

    		 —

    返回值

    类型：boolean

    如果 obj 是正则则返回 true，否则返回 false。

    示例

    ❐isRegExp(/a/) // true
    ❐isRegExp("/a/") // false

    判断一个对象是否是正则。
    ⅏ ⮞ join(...)(...regexps:RegExp[]):RegExp

    合并正则并选择其一匹配。

    参数 类型 描述 默认值
    regexps RegExp[]

    要合并的正则列表。

    		 —

    返回值

    类型：RegExp

    返回一个新正则表达式对象。

    合并正则并选择其一匹配。
    ⅏ ⮞ fromWildcard(value)(value:string):RegExp

    将一个通配符表达式转为等效正则表达式。

    参数 类型 描述 默认值
    value* string

    要转换的通配符表达式。

    		 —

    返回值

    类型：RegExp

    返回转换后的等效正则表达式。

    说明

    通配符表达式意义： 其中 * 表示匹配任意字符。 其中 ? 表示匹配一个字符。 如果路径以 / 开头则匹配跟目录。否则匹配任意路径部分。

    示例

    ❐fromWildcard("a*b").test("ab") // true
    ❐fromWildcard("a*b").test("acb") // true
    ❐fromWildcard("a*b").test("acbd") // false

    将一个通配符表达式转为等效正则表达式。
