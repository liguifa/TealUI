    util/path
  • 复制 import "util/path"
  • 复制 require("util/path")
  • 复制 "util/path"

    提供处理文件路径的工具函数。

    ☍API

    函数 描述
    ⅏ ⮞ isAbsolute(path)(path:string):boolean

    判断指定的路径是否为绝对路径。

    参数 类型 描述 默认值
    path* string

    判断的路径。

    		 —

    返回值

    类型：boolean

    如果是绝对路径则返回 true，否则返回 false。

    参考

    • https://github.com/joyent/node/blob/master/lib/path.js

    判断指定的路径是否为绝对路径。
    ⅏ ⮞ resolve(...)(...paths:string[]):string

    合并多个路径为一个。

    参数 类型 描述 默认值
    paths string[]

    要合并的路径。

    		 —

    返回值

    类型：string

    返回合并后的新路径。

    示例

    ❐Path.resolve("a/b", "../", "c") // "a/c"

    合并多个路径为一个。
    ⅏ ⮞ relative(basePath, path)(basePath:string, path:string):string

    计算指定路径相对于基路径的相对路径。

    参数 类型 描述 默认值
    basePath* string

    解析的基路径。

    		 —
    path* string

    要处理的路径。

    		 —

    返回值

    类型：string

    返回 path 相对于 basePath 的基路径。

    示例

    ❐Path.relative("a/b", "a/c") // "../c"

    计算指定路径相对于基路径的相对路径。
    ⅏ ⮞ normalize(path)(path:string):string

    规划化指定的路径。

    参数 类型 描述 默认值
    path* string

    要处理的路径。

    		 —

    返回值

    类型：string

    返回处理后的新路径。

    示例

    ❐Path.normalize("a/b/../c/d/e") // "a/c/d/e"

    规划化指定的路径。
    ⅏ ⮞ join(...)(...paths:string[]):string

    合并多个文件夹路径为一个。

    参数 类型 描述 默认值
    paths string[]

    要处理的文件夹路径。

    		 —

    返回值

    类型：string

    返回合并后的新路径。

    示例

    ❐Path.join("a/b/../c/d/e") // "a/c/d/e"

    合并多个文件夹路径为一个。
    ⅏ ⮞ dirname(path)(path:string):string

    获取指定路径的文件夹名部分。

    参数 类型 描述 默认值
    path* string

    要处理的路径。

    		 —

    返回值

    类型：string

    返回文件夹部分。

    示例

    ❐Path.dirname("e/a/b") // "e/a"

    获取指定路径的文件夹名部分。
    ⅏ ⮞ basename(path, ...)(path:string, ext?:string):string

    获取指定路径的文件名部分。

    参数 类型 描述 默认值
    path* string

    要处理的路径。

    		 —
    ext string | undefined

    如果指定扩展名，则删除对应的扩展名部分。

    		 —

    返回值

    类型：string

    返回文件部分。

    示例

    ❐Path.basename("e/a/b.txt") // "b.txt"

    获取指定路径的文件名部分。
    ⅏ ⮞ extname(path)(path:string):string

    获取指定路径的扩展名部分（包括点）。

    参数 类型 描述 默认值
    path* string

    要处理的路径。

    		 —

    返回值

    类型：string

    返回扩展名部分。

    示例

    ❐Path.extname("e/a/b.txt") // ".txt"

    获取指定路径的扩展名部分（包括点）。
