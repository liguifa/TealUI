    ☍常量

    模板内任意字符串都会原样输出，模板引擎只解析 {} 内的数据。模板内使用 {{ 代替 {，使用 }} 代替 }

    ☍逻辑

    所有的表达式语法和 JavaScript 相同。

    if 语句

            {if (true)}
            text1
        {else if (false)}
            text2
        {else}
            text3
        {/if}

    foreach 语句

            {foreach (value, key in [0,1,2])}
            {key} = {value}
        {/foreach}

    foreach 语句同时支持数组和对象。

    for 语句

            {for (var i=0; i < 3; i++) }
            text{i}
        {/for}
        {for (var key in obj) }
            text{key}
        {/for}

    while 语句

            {while (false) }
            text
        {/while}

    var 语句

            {var a = 2;}

    function 语句

            {function a()}
            func
        {/function}
        {a() // 函数调用}

    void 语句

    void 语句用于执行代码，但不会在模板字符串内添加任何内容。

            {void alert("alert")}

    ☍内置宏变量

    在模板内部可以直接使用一些内置宏变量。

    $data

    被解析的数据。

    $key

    foreach 语句中获取最近的循环索引或键。

    $target

    foreach 语句中获取最近的循环对象。
