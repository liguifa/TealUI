    ⅏ ⮞ toRGB(value)(value:Color):RGB

    将任意颜色转为 RGB 格式。

    参数 类型 描述 默认值
    value* Color —

    返回值

    类型：RGB

    返回包含 RGB 信息的对象。

    示例

    ❐toRGB("#000") // {r: 0, g: 0, b: 0}
    ❐toRGB("#ff0000") // {r: 255, g: 0, b: 0}
    ❐toRGB("#ff0000ff") // {r: 255, g: 0, b: 0, a: 1}
    ❐toRGB("rgb(255,0,0)") // {r: 255, g: 0, b: 0, a: 1}
    ❐toRGB("rgba(255,0,0,0.9)") // {r: 255, g: 0, b: 0, a: 0.9}
    ❐toRGB("hsl(255,0,0)") // {r: 255, g: 0, b: 0, a: 0.9}
    ❐toRGB("hsla(255,0,0,0.9)") // {r: 255, g: 0, b: 0, a: 0.9}
    ❐toRGB({h:255, s: 0, l: 0}) // {r: 255, g: 0, b: 0}
    ❐toRGB({h:255, s: 0, l: 0, a: 0}) // {r: 255, g: 0, b: 0}

    将任意颜色转为 RGB 格式。
    ⅏ ⮞ toHSL(value)(value:Color):void

    将任意颜色转为 HSL 格式。

    参数 类型 描述 默认值
    value* Color —

    返回值

    类型：void

    返回包含 HSL 信息的对象。

    示例

    ❐toHSL("#000") // {h: 0, s: 0, l: 0}
    ❐toHSL("#ff0000") // {h: 0, s: 255, l: 128}
    ❐toHSL("#ff0000ff") // {h: 0, s: 255, l: 128, a: 1}
    ❐toHSL("rgb(255,0,0)") // {h: 0, s: 255, l: 128}
    ❐toHSL("rgba(255,0,0,1)") // {h: 0, s: 255, l: 128, a: 1}
    ❐toHSL("hsl(0, 255, 128)") // {h: 0, s: 255, l: 128}
    ❐toHSL("hsla(0, 255, 128, 1)") // {h: 0, s: 255, l: 128, a: 1}
    ❐toHSL({r:255, g: 0, b: 0}) // {h: 0, s: 255, l: 128}
    ❐toHSL({r:255, g: 0, b: 0, a: 1}) // {h: 0, s: 255, l: 128, a: 1}

    将任意颜色转为 HSL 格式。
