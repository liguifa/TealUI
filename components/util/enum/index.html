    ☍枚举
    util/enum
  • 复制 import "util/enum"
  • 复制 require("util/enum")
  • 复制 "util/enum"
    •

    枚举即将常用的字段用统一的数字编码存储表示，程序中使用字段名指代难记的数字。定义枚举可增加代码的可读性。

    定义枚举
    定义枚举标记位
    API

    ☍定义枚举

    ❐var WeekDay = { 
    sunday: 0,
    monday: 1,
    tuesday: 2,
    wednesday: 3,
    thirsday: 4,
    friday: 5,
    saturday: 6
};

    ☍定义枚举标记位

    通过位操作将多个标记存在同一个整数中。

    ❐var Colors = {
    red: 1 << 0,
    yellow: 1 << 1,
    blue: 1 << 2
};
var green = Colors.red | Colors.yellow;

    ☍API

    函数 描述
    ⅏ ⮞ getNames(enumType)(enumType:object):string[]

    获取枚举类型中定义的字段。

    参数 类型 描述 默认值
    enumType* object

    枚举类型对象。

    		 —

    返回值

    类型：string[]

    返回所有键组成的数组。

    获取枚举类型中定义的字段。
    ⅏ ⮞ getName(enumType, enumValue)(enumType:any, enumValue:any):string

    根据枚举值返回其枚举名。

    参数 类型 描述 默认值
    enumType* any

    枚举类型对象。

    		 —
    enumValue* any

    枚举的内容。

    		 —

    返回值

    类型：string | undefined

    返回枚举键。如果不存在则返回 undefined。

    示例

    ❐getName(WeekDay, 0) // 'sunday'

    根据枚举值返回其枚举名。
    ⅏ ⮞ hasFlag(enumValue, flag)(enumValue:number, flag:number):boolean

    判断指定枚举值是否包含指定的标记位。

    参数 类型 描述 默认值
    enumValue* number

    要判断的枚举值。

    		 —
    flag* number —

    返回值

    类型：boolean

    如果包含标记位则返回 true，否则返回 false。

    示例

    ❐var Flags = {
red: 1 << 1,
yellow: 1 << 2,
blue: 1 << 3,
};
Enum.hasFlag(Flags.red | Flags.yellow, Flags.red) // true

    判断指定枚举值是否包含指定的标记位。
    ⅏ ⮞ setFlag(enumValue, flag, value)(enumValue:number, flag:number, value:boolean):number

    设置指定枚举的标记位。

    参数 类型 描述 默认值
    enumValue* number

    要判断的枚举值。

    		 —
    flag* number —
    value* boolean

    如果设置为 true 表示添加标记位，否则清空标记位。

    		 —

    返回值

    类型：number

    返回更新后的枚举值。

    示例

    ❐var Flags = {
red: 1 << 1,
yellow: 1 << 2,
blue: 1 << 3,
};
var flag = 0;
flag = Enum.setFlag(flag, Flags.red, true)

    设置指定枚举的标记位。
