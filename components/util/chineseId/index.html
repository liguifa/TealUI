    中国身份证号处理

    ❐<input type="text" id="input" placeholder="输入身份证号" value="152500198909267865" />
<button onclick="var idInfo = parseChineseId(input.value); output.innerHTML = '合法：' + idInfo.valid + '<br>性别：' + (idInfo.sex ? '男' : '女') + '<br>省份：' + (idInfo.province || '') + '<br>生日： ' + idInfo.birthday.toLocaleString();">校验中国身份证</button>
<div id="output"></div>
    ⏷

    ☍API

    函数 描述
    ⅏ ⮞ parseChineseId(value)(value:string):object

    解析身份证号的信息并检验合法性。

    参数 类型 描述 默认值
    value* string

    要解析的身份证号。

    		 —

    返回值

    类型：{ valid: boolean; province: string; birthday: Date; sex: boolean; }

    返回一个 JSON，如：

    ❐{
"valid": true,      // 表示身份证信息合法。
"province": "北京", // 表示省份。
"birthday": Date(), // 表示生日。
"sex": false        // 表示性别。true 表示 '男'。
}

    说明

    本函数只验证身份证的数值特征，因此只适用于过滤无效的身份证号，并不能判定身份证是否真实存在。

    示例

    ❐parseChineseId("152500198909267865")

    解析身份证号的信息并检验合法性。
