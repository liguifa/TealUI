    ☍货币处理
    util/currency
  • 复制 import "util/currency"
  • 复制 require("util/currency")
  • 复制 "util/currency"
    •

    提供货币精确运算函数。

    注意

    货币组件支持计算的最大货币为：10,000,000,000,000.00，如需支持更大范围，请使用大数组件。

    ☍API

    函数 描述
    ⅏ ⮞ add(x, y)(x:number, y:number):number

    精确计算两个货币的和。

    参数 类型 描述 默认值
    x* number

    计算的左值，最大不得超过 1 万亿，最多只能有两位小数。

    		 —
    y* number

    计算的右值，最大不得超过 1 万亿，最多只能有两位小数。

    		 —

    返回值

    类型：number

    返回计算的结果。

    示例

    ❐add(86.24, 0.1) // 86.34

    精确计算两个货币的和。
    ⅏ ⮞ sub(x, y)(x:number, y:number):number

    精确计算两个货币的差。

    参数 类型 描述 默认值
    x* number

    计算的左值，最大不得超过 1 万亿，最多只能有两位小数。

    		 —
    y* number

    计算的右值，最大不得超过 1 万亿，最多只能有两位小数。

    		 —

    返回值

    类型：number

    返回计算的结果。

    示例

    ❐sub(7, 0.8) // 6.2

    精确计算两个货币的差。

    ⅏ ⮞ round(value)(value:number):number

    保留小数点后两位四舍五入。

    参数 类型 描述 默认值
    value* number

    要处理的值。

    		 —

    返回值

    类型：number

    返回计算的结果。

    示例

    保留小数点后两位四舍五入。
    ⅏ ⮞ format(value)(value:number):string

    将指定的货币格式化为字符串。

    参数 类型 描述 默认值
    value* number

    要处理的值。

    		 —

    返回值

    类型：string

    返回格式化后的字符串。字符串保留两位小数，整数部分每 3 位包含一个逗号分隔符。

    示例

    ❐format(86234.245) // "86,234.25"

    将指定的货币格式化为字符串。
