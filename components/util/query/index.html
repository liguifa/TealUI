    解析和格式化查询字符串（如：?a=1&b=2）

    ☍API

    函数 描述
    ⅏ ⮞ parseQuery(value, ...)(value:string, joinChar?:string, equalChar?:string):object

    解析一个查询字符串为对象。

    参数 类型 描述 默认值
    value* string

    要解析的查询字符串。

    		 —
    joinChar string

    使用的连接符。

    		 "&"
    equalChar string

    使用的等号。

    		 "="

    返回值

    类型：{ [key: string]: any; }

    返回解析后的对象。

    示例

    ❐parseQuery("foo=1&goo=2&goo=3") // { foo: "1", goo: ["2", "3"] }

    解析一个查询字符串为对象。
    ⅏ ⮞ formatQuery(obj, ...)(obj:any, joinChar?:string, equalChar?:string):string

    将指定对象格式化为查询字符串。

    参数 类型 描述 默认值
    obj* any

    要格式化的对象。

    		 —
    joinChar string

    使用的连接符。

    		 "&"
    equalChar string

    使用的等号。

    		 "="

    返回值

    类型：string

    返回格式化后的字符串。

    示例

    ❐formatQuery({ a: "2", c: "4" }) // "a=2&c=4"
    ❐formatQuery({ a: [2, 4] }) // "a=2&a=4"

    将指定对象格式化为查询字符串。
    ⅏ ⮞ getQuery(name, ...)(name:string, url?:string):string

    获取指定地址的查询参数值。

    参数 类型 描述 默认值
    name* string

    要获取的查询参数名。

    		 —
    url string

    要处理的地址，默认为当前页面的地址。

    		 location.href

    返回值

    类型：string | null

    返回查询参数值。如果获取不到则返回 null。

    示例

    ❐getQuery("foo", "?foo=1") // "1"
    ❐getQuery("goo", "?foo=1") // null

    获取指定地址的查询参数值。
    ⅏ ⮞ setQuery(name, value, ...)(name:string, value:string, url?:string):string

    设置指定的地址的查询参数值。

    参数 类型 描述 默认值
    name* string

    要设置的查询参数名。

    		 —
    value* string | null

    要设置的查询参数值。如果设为 null 则删除。

    		 —
    url string

    要处理的地址，默认为当前页面的地址。

    		 location.href

    返回值

    类型：string

    返回更新后的地址。

    示例

    ❐setQuery("foo", "1", "page.html") // "page.html?foo=1"
    ❐setQuery("foo", "2", "page.html?foo=1") // "page.html?foo=2"
    ❐setQuery("foo", null, "page.html") // "page.html"

    设置指定的地址的查询参数值。
