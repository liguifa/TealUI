    ☍API

    函数 描述
    ⅏ ⮞ checkPassword(value)(value:string):number

    检验密码的复杂度。

    参数 类型 描述 默认值
    value* string

    要检查的密码。

    		 —

    返回值

    类型：number

    返回一个整数。值越大表示复杂度越高。具体范围如下：

    • 复杂度 < 0：简单密码。
    • 复杂度 = 0：普通密码。
    • 复杂度 > 0：复杂密码。
    • 复杂度 > 2：安全密码。

    说明

    密码检验策略：

    1. 将字符分为四类：数字，小写字母，大写字母，特殊字符。种类转换次数越多越复杂。
    2. 重复字符、连续字符越多越简单。

    示例

    ❐checkPassword("123456") // -1
