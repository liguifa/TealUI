    ⅏ ⮞ bubbleSort<T>(iteratable, ...)(iteratable:ArrayList<T>, comparer?:function, start?:number, end?:number):ArrayList<T>

    对类数组进行冒泡排序。

    泛型参数 约束类型 默认类型
    T — —
    参数 类型 描述 默认值
    iteratable* ArrayList<T>

    要排序的类数组。

    		 —
    comparer (x: T, y: T) => boolean

    用于排序时确定优先级的函数。函数的参数为：

    • x：比较的第一个参数。
    • y：比较的第二个参数。
    • 如果返回 true，则说明 x 应该排在 y 之前。否则 x 应该排在 y 之后。
    		 (x: T, y: T) => x < y

    end number

    结束排序的索引。

    返回值

    类型：ArrayList<T>

    说明

    冒泡排序是稳定排序算法。

    示例

    ❐bubbleSort([1, 3, 5, 4, 3]) // [1, 3, 3, 4, 5]

    对类数组进行冒泡排序。
