    DES 加密和解密实现。

    ❐<input type="text" id="input" placeholder="输入任意内容" /> 
<input type="text" id="key" placeholder="输入加解密的键" value="key" /> 
<button onclick="input.value = des(input.value, key.value)" />DES 加密</button>
<button onclick="input.value = des(input.value, key.value, true)">DES 解密</button>
    ⏷

    ☍API

    函数 描述
    ⅏ ⮞ des(value, key, ...)(value:string, key:string, decrypt?:boolean, iv?:string, padding?:number):string

    使用 DES 算法加密或解密指定字符串。

    参数 类型 描述 默认值
    value* string

    要加密或解密的字符串。

    		 —
    key* string

    加密或解密使用的密钥。

    		 —
    decrypt boolean

    如果是 true 则执行解密，否则执行加密(默认)。

    		 false
    iv string | undefined

    如果使用密码段链接（Cipher Block Chaining，CBC）模式，则提供初始向量；如果未提供则不使用此模式。

    		 —
    padding number

    自动填充的字符数，可以是 0、1 或 2。

    		 0

    返回值

    类型：string

    返回加密后的字符串。

    示例

    ❐des("a", "1") // "\u0082\u000e\u0056\u00cc\u007c\u0045\u0059\u00a4"
