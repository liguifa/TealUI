    ☍Cookie 处理扩展
    ux/cookie/cookie-more

    提供列出 Cookie 和子 Cookie 功能。

    ☍API

    函数 描述
    ⅏ ⮞ getAllCookies()():object

    获取所有 Cookie 。

    返回值

    类型：{ [key: string]: string; }

    返回包含所有 Cookie 的键值对。

    示例

    ❐getAllCookies()

    获取所有 Cookie 。
    ⅏ ⮞ getSubcookie(name, subname)(name:string, subname:string):string

    获取一个子 Cookie 值。

    参数 类型 描述 默认值
    name* string

    要获取的 Cookie 名字。

    		 —
    subname* string

    要获取的子 Cookie 名字。

    		 —

    返回值

    类型：string | null

    返回对应的 Cookie 值。如果 Cookie 不存在则返回 null。

    示例

    ❐getSubcookie("sample", "subName") // 如果不存在则返回 null。

    获取一个子 Cookie 值。
    ⅏ ⮞ setSubcookie(name, subname, value, ...)(name:string, subname:string, value:string, expires?:number, path?:string, domain?:string, secure?:string):void

    设置或删除一个子 Cookie 值。

    参数 类型 描述 默认值
    name* string

    要设置的 Cookie 名字。

    		 —
    subname* string

    要设置的子 Cookie 名字。

    		 —
    value* string | null

    要设置的 Cookie 值。如果设为 null 则删除 Cookie。

    		 —
    expires number | undefined

    Cookie 过期的秒数。如果设为 -1 表示永不过期。如果设为 0 表示立即过期。

    		 —
    path string | undefined

    设置 Cookie 的路径。

    		 —
    domain string | undefined

    设置 Cookie 的所在域。

    		 —
    secure string | undefined

    设置 Cookie 的安全限制。

    		 —

    返回值

    类型：void

    示例

    ❐// 设置子 Cookie
setSubcookie("sample", "subName", "the value")

// 删除子 Cookie
setSubcookie("sample", "subName", null)

    设置或删除一个子 Cookie 值。
