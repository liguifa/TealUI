    ⅏ ⮞ setCookie(name, value, ...)(name:string, value:string, expires?:number, path?:string, domain?:string, secure?:string):void

    设置或删除一个 Cookie。

    参数 类型 描述 默认值
    name* string

    要设置的 Cookie 名。

    		 —
    value* string | null

    要设置的 Cookie 值。如果设为 null 则删除 Cookie。

    		 —
    expires number

    Cookie 过期的秒数。如果设为 -1 表示永不过期。如果设为 0 表示立即过期。

    		 365 * 24 * 60 * 60 * 10

    设置 Cookie 的路径。

    		 —
    domain string | undefined

    设置 Cookie 的所在域。

    		 —
    secure string | undefined

    设置 Cookie 的安全限制。

    		 —

    返回值

    类型：void

    示例

    ❐// 设置 Cookie
setCookie("sample", "the value")
