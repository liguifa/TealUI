    ☍图片懒加载
    ux/lazyLoad
  • 复制 import "ux/lazyLoad"
  • 复制 require("ux/lazyLoad")
  • 复制 "ux/lazyLoad"
    •

    仅当滚到图片时才加载，减少请求数。

    ❐<img id="img" src="../../../assets/resources/100x100.png" data-src="../../../assets/resources/200x200.png">
<script>
import lazyLoad from "ux/lazyLoad";

lazyLoad(img, img.getAttribute("data-src"));
</script>

    ☍API

    函数 描述
    ⅏ ⮞ lazyLoad(elem, url, ...)(elem:HTMLImageElement, url:string, scrollParent?:HTMLElement, callback?:function):void

    懒加载图片。

    参数 类型 描述 默认值
    elem* HTMLImageElement

    要加载的图片。

    		 —
    url* string

    加载的地址。

    		 —
    scrollParent HTMLElement | undefined

    可滚动的元素。

    		 —
    callback (() => void) | undefined

    图片已加载的回调函数。

    		 —

    返回值

    类型：void

    示例

    ❐lazyLoad(img)
