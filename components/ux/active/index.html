    快速切换当前选中的元素。

    目的
    API

    ☍目的

    项目中一般通过添加 .active CSS 类名表示选中。使用 setActive 可以快速切换：

    ❐setActive([elem1, elem2, elem3], "active", elem2);

// 等效于以下代码：
// removeClass(elem1, "active");
// addClass(elem2, "active");
// removeClass(elem3, "active");

    ☍API

    函数 描述
    ⅏ ⮞ getActive(items, className)(items:ArrayLike<HTMLElement>, className:string):any

    获取列表中的元素的激活元素。

    参数 类型 描述 默认值
    items* ArrayLike<HTMLElement>

    所有元素列表。

    		 —
    className* string

    激活的 CSS 类名。

    		 —

    返回值

    类型：any

    返回激活的元素。如果不存在则返回 null。

    获取列表中的元素的激活元素。
    ⅏ ⮞ setActive(items, className, value)(items:ArrayLike<HTMLElement>, className:string, value:HTMLElement):void

    设置列表中指定的元素为激活样式。

    参数 类型 描述 默认值
    items* ArrayLike<HTMLElement>

    所有元素列表。

    		 —
    className* string

    激活的 CSS 类名。

    		 —
    value* HTMLElement | null

    要激活的元素。如果为 null 则表示不再激活新元素。

    		 —

    返回值

    类型：void

    设置列表中指定的元素为激活样式。
