    实现网页倒计时和正计时效果。

    倒计时
    隐藏天数
    正计时
    API

    ☍倒计时

    ❐import countDown from "ux/countDown";

countDown(new Date('2020/1/1'), (days, hours, minutes, seconds, total) => {
    __root__.innerHTML = `还有${days}天${hours}小时${minutes}分${seconds}秒`;
});

    ☍隐藏天数

    ❐import countDown from "ux/countDown";

countDown(new Date('2020/1/1'), (days, hours, minutes, seconds, total) => {
    __root__.innerHTML = `还有${+days * 24 + +hours}小时${minutes}分${seconds}秒`;
});

    ☍正计时

    ❐import { countUp } from "ux/countDown";

countUp(new Date('2011/8/12'), (years, days, hours, minutes, seconds, total) => {
    __root__.innerHTML = `TealUI 已发布：${years}年${days}天${hours}小时${minutes}分${seconds}秒`;
});

    ☍API

    函数 描述
    ⅏ ⮞ countDown(end, callback, ...)(end:Date, callback:function, now?:Date):number

    执行倒计时。

    参数 类型 描述 默认值
    end* Date

    倒计时的结束时间。

    		 —
    callback* (days: string, hours: string, minutes: string, seconds: string, total: number) => boolean | void

    每隔一秒执行的回调函数，可以用于更新界面。函数的参数依次为:

    • days：需要显示的剩余天数。
    • hours：需要显示的剩余小时数。
    • minutes：需要显示的剩余分数。
    • seconds：需要显示的剩余秒数。
    • total：剩余总秒数。
    		 —
    now Date

    当前时间。可以由服务器下发以避免用户设置时间的误差。

    		 new Date()

    返回值

    类型：number

    返回一个计时器，可以通过 clearInterval() 停止倒计时。

    示例

    ❐countDown(new Date("2020/1/1"), (day, hour, minute, second) => console.log(day, hour, minute, second))

    执行倒计时。
