    ☍强制打开
    ux/forceOpen
  • 复制 import "ux/forceOpen"
  • 复制 require("ux/forceOpen")
  • 复制 "ux/forceOpen"
    •

    防止新开窗口被浏览器拦截。

    ❐<button onclick="forceOpen('http://tealui.com/')">强制打开网页</button>
    原理

    默认地，在点击事件中新开窗口不会被浏览器拦截。 如果发现新窗口被拦截，则捕获首次点击事件强制打开页面。

    ☍API

    函数 描述
    ⅏ ⮞ forceOpen(url)(url:string):void

    强制打开指定网页。

    参数 类型 描述 默认值
    url* string

    要打开的地址。

    		 —

    返回值

    类型：void

    示例

    ❐forceOpen("http://tealui.com/")

    强制打开指定网页。
