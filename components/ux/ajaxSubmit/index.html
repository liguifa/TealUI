    使用 AJAX 异步提交表单。

    ☍API

    函数 描述
    ⅏ ⮞ ajaxSubmit(formElem, ...)(formElem:HTMLFormElement, success?:function, error?:function, options?:Partial<Ajax>):Promise<{}>

    Ajax 异步提交表单。

    参数 类型 描述 默认值
    formElem* HTMLFormElement

    要提交的表单元素。

    		 —
    success ((responseText: string, ajax: Ajax) => void) | undefined

    成功回调函数。

    		 —
    error ((error: any, ajax: Ajax) => void) | undefined

    错误回调函数。

    		 —
    options Partial<Ajax> | undefined

    附加的额外选项。

    		 —

    返回值

    类型：Promise<{}> | undefined

    如果未设置任何回调函数则返回一个确认对象。

    示例

    ❐ajaxSubmit(document.getElementById("form"))

    Ajax 异步提交表单。
    ⅏ ⮞ formData(formElem, ...)(formElem:HTMLFormElement, disabled?:boolean):FormData | object

    获取表单的数据。

    参数 类型 描述 默认值
    formElem* HTMLFormElement

    表单元素。

    		 —
    disabled boolean

    是否包含被禁用的元素。

    		 false

    返回值

    类型：FormData | { [key: string]: string | string[]; }

    返回表单数据。

    示例

    ❐formData(document.getElementById("form"))

    获取表单的数据。
