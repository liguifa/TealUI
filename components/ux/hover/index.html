    ☍鼠标移入事件
    ux/hover
  • 复制 import "ux/hover"
  • 复制 require("ux/hover")
  • 复制 "ux/hover"
    •

    模拟 CSS :hover 伪类。

    ❐<div id="box" class="doc-box"></div>
<script>
import hover from "ux/hover";

hover(box, e => {
    box.className += " doc-box-blue";
}, e => {
    box.className = "doc-box";
})
</script>

    ☍API

    函数 描述
    ⅏ ⮞ hover(elem, ...)(elem:HTMLElement, mouseEnter?:function, mouseLeave?:function, delay?:number):void

    绑定鼠标移上后的操作。

    参数 类型 描述 默认值
    elem* HTMLElement

    相关的元素。

    		 —
    mouseEnter ((e: MouseEvent) => void) | undefined

    鼠标移入后的操作。

    		 —
    mouseLeave ((e: MouseEvent) => void) | undefined

    鼠标移出后的操作。

    		 —
    delay number

    触发事件延时执行的毫秒数。鼠标进入后指定时间内不触发函数。

    		 30

    返回值

    类型：void

    示例

    ❐hover(elem, function(){ alert("鼠标进来了") }, function(){ alert("鼠标出去了") });

    绑定鼠标移上后的操作。
