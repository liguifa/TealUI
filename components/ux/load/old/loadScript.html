    ⅏ ⮞ loadScript(url, callback)(url:string, callback:any):HTMLScriptElement

    动态载入一个 JavaScript 脚本。

    参数 类型 描述 默认值
    url* string

    加载 JavaScript 文件的路径。

    		 —
    callback* any

    载入成功的回调函数。函数参数为： *

    		 —

    返回值

    类型：HTMLScriptElement

    返回用于载入脚本的 <script> 标签。

    示例

    ❐{String} url 加载的 JavaScript 文件路径。
    ❐loadScript("../../assets/resources/ajax/test.js")
