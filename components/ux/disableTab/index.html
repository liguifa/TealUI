    ☍禁止 Tab
    ux/disableTab
  • 复制 import "ux/disableTab"
  • 复制 require("ux/disableTab")
  • 复制 "ux/disableTab"
    •

    禁止 Tab 切换焦点并改成输入。

    ❐<input type="text" id="input" placeholder="按 Tab">
<script>
    import disableTab from "ux/disableTab";
    
    disableTab(input);
</script>

    ☍API

    函数 描述
    ⅏ ⮞ disableTab(elem, ...)(elem:HTMLInputElement, tab?:string):void

    禁止 Tab 切换焦点，改为输入 Tab。

    参数 类型 描述 默认值
    elem* HTMLInputElement

    相关的输入框。

    		 —
    tab string

    输入的 Tab 符号。

    		 "\t"

    返回值

    类型：void

    禁止 Tab 切换焦点，改为输入 Tab。
