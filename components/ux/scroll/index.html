    控制元素的滚动条。

    ☍API

    函数 描述
    ⅏ ⮞ scrollTo(elem, value, ...)(elem:HTMLElement | Document, value:Partial<Point>, duration?:number, callback?:function):void

    滚动元素到指定的位置。

    参数 类型 描述 默认值
    elem* HTMLElement | Document

    要滚动的元素或文档。

    		 —
    value* Partial<Point>

    要滚动的目标坐标。允许只设置部分属性。

    		 —
    duration number

    滚动的动画时间。如果为 0 则不使用动画。

    		 200
    callback ((value: Point) => void) | undefined

    滚动动画结束的回调函数。

    		 —

    返回值

    类型：void

    示例

    ❐scrollTo(document, {y: 400})

    滚动元素到指定的位置。
