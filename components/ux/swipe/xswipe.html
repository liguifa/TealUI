    ⅏ ⮞ XSwipe(elem, interval, touchMove)(elem:any, interval:any, touchMove:any):void

    将容器直接子节点作为列表进行自动轮播。

    参数 类型 描述 默认值
    elem* any

    滚动的容器节点。一般是

      元素。

    		 —
    interval* any

    =0 自动轮播的间隔时间，如果为 0 则不自动轮播。

    		 —
    touchMove* any

    =true 是否启用滑动轮播。

    		 —

    返回值

    类型：void

    说明

    限制：

    • 只能在 Webkit 内核的手机浏览器使用。
    • 所有子节点必须和容器等宽。
    • 一次只能移动一张。
