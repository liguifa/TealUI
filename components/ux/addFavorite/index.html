    ☍添加到收藏夹
    ux/addFavorite
  • 复制 import "ux/addFavorite"
  • 复制 require("ux/addFavorite")
  • 复制 "ux/addFavorite"
    •

    引导用户收藏当前网页。

    ☍API

    函数 描述
    ⅏ ⮞ addFavorite(...)(title?:string, url?:string):void

    打开添加收藏夹对话框。

    参数 类型 描述 默认值
    title string

    收藏的标题。默认为当前网页标题。

    		 document.title
    url string

    收藏的地址。默认为当前网页地址。

    		 location.href

    返回值

    类型：void

    说明

    最新浏览器由于安全限制，不允许使用此功能。这时，函数会提示用户手动操作。

    示例

    ❐// 添加当前网页到收藏夹
addFavorite()

// 添加指定网页到收藏夹
addFavorite("TealUI", "http://tealui.com")

    打开添加收藏夹对话框。
    ⅏ ⮞ setHomePage(url)(url:string):void

    弹出设置主页对话框。

    参数 类型 描述 默认值
    url* string

    设置的地址。

    		 —

    返回值

    类型：void

    说明

    最新浏览器由于安全限制，不允许使用此功能。这时，函数会提示用户手动操作。

    示例

    ❐// 设置当前网页为主页
setHomePage()

// 设置指定网页为主页
setHomePage("TealUI", "http://tealui.com")

    弹出设置主页对话框。
