语法介绍

常量

模板内任意字符串都会原样输出，模板引擎只解析 {} 内的数据。模板内使用 {{ 代替 {，使用 }} 代替 }

逻辑

所有的表达式语法和 JavaScript 相同。

if 语句

foreach 语句

foreach 语句同时支持数组和对象。

for 语句

注意 foreach 和 for 语句的区别是：foreach 没有括号；for 语句是JavaScript 自带的语法，需要括号。

while 语句

var 语句

function 语句

void 语句

void 语句用于执行代码，但不会在模板字符串内添加任何内容。

内置宏变量

在模板内部可以直接使用一些内置宏变量。

$data

被解析的数据。

$key

for each 语句中获取最近的循环索引或键。

$target

for each 语句中获取最近的循环对象。
