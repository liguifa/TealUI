功能齐全：覆盖前端多数需求；同时支持电脑和手机！

TealUI 包含了一套简约现代风格的 CSS 设计、一套高性能的 JS 工具代码库和一套现成的完整网页模板。你既可以通过复制源码的方式单独使用某个功能，也可以整体打包下载作为 UI 框架使用。

无需设计：快速搭建漂亮页面

100 多个相同风格的 UI 设计，让工(ji)程(shu)师(zhai)无需设计、用最少代码就能搭建出漂亮的 HTML5 页面，自动适配电脑手机多个平台。

工具函数：轻松复制现成源码

如日期扩展、Cookie 处理、MD5 加密、模板引擎。这些常用函数都能在库中找到，函数代码短、依赖少，无需下载，只要复制源码就能直接使用。

更懂国情：适合中文排版

根据国内业务需要，提供城市选择、拼音汉字转换、身份证验证等特定组件；兼顾国内行情，为国产浏览器填坑。
