无需设计：快速搭建漂亮页面；同时支持电脑和手机！

TealUI 提供了 100 多个风格统一的 UI 设计，让工(ji)程(shu)师(zhai)无需设计、用最少代码就能搭建出漂亮的 HTML5 页面。所有 UI 组件都可单独使用，复制示例代码即可见效。

轻量组件化思路

独创轻量组件化开发思路可让组件独立使用；不写 JS 实现交互功能；精致组件底层构架更让组件用法统一、扩展方便。

功能齐全：不只是一个 UI 库

300 个组件、4000 个 API、60000 行代码，覆盖前端多数需求。如 Cookie 处理、MD5 加密、模板引擎都能在库中找到。

更懂国情：适合中文排版

根据国内业务需要，提供城市选择、拼音汉字转换、身份证验证等特定组件；兼顾国内行情，为国产浏览器填坑。
